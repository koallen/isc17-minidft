# miniDFT

## Compilation

There are 2 variables, `arch` and `comp` (compiler).

### Architecture

To compile CPU version, use

```bash
$ make arch=cpu
```

To compile GPU version, use

```bash
$ make
```

### Compiler

To compile with Intel, use

```bash
$ make comp=intel
```

To compile with PGI, use

```bash
$ make
```
